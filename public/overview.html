Github link: https://github.com/pmservice/drug-selection

This sample application adopts SPSS Modeler stream Drug1N.str, Watson Machine Learning service on Bluemix platform for drug selection.

With this application you are able to:

  • provide patient personal data and patient blood test result using web application form
  • run scoring model by pressing "Check Now" button
  • display information about recommended drug (scoring result) in form of table
